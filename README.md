## 🛠️ Featured Projects

[![WorkPortal](https://svg.bookmark.style/api?url=https://github.com/picoder15/workportal&mode=dark&style=horizontal)](https://github.com/picoder15/workportal)

**WorkPortal** is a modern, self‑hosted knowledge management system that works like a personal wiki.  
It comes with:
- A rich‑text editor (with internal linking, images, and version history)
- A visual graph of connected notes
- A dashboard with calendar and activity timeline  
- Cross‑device sync over your local network  
- **Tech Stack:** React 19, TypeScript, Vite, Tailwind CSS, Node.js, Express

[![Tmonitor](https://svg.bookmark.style/api?url=https://github.com/picoder15/Tmonitor&mode=dark&style=horizontal)](https://github.com/picoder15/Tmonitor)

**Tmonitor** is a system monitoring tool designed to keep an eye on your infrastructure.  
- Built with TypeScript and Python  
- Focus on simplicity and real‑time insights  
- Actively under development
